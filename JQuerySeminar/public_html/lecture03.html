Review

The following are true statements about obj = new Foo();

  1. obj.__proto__ === Foo.prototype
  2. Let x be a function in obj.__proto__.
    1. x is executed by calling obj.x().
    2. x almost certainly uses "this" somewhere.
    3. The value of "this" during the execution of x will be obj.
    4. obj.x = function(){} will override the x in obj.__proto__.

Review

The following are true statements about

   var obj = $("p");
   obj.html("hello");
  1. obj.length will equal the number of <p> elements.
  2. obj[n] will be the DOM element for the nth <p> element.
  3. obj.__proto__ === jQuery.prototype
  4. html is a function in obj.__proto__.
    1. html is executed by calling obj.html("hello").
    2. The value of "this" during the execution of html will be obj.
    3. html uses "this" to access the DOM elements of obj.
    4. obj.html = function(){} will override the html in obj.__proto__. Affects obj only. Will not affect future calls to new jQuery().
    5. jQuery.prototype.html = function(){} will affect obj and all future calls to new jQuery().
