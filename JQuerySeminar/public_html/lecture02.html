Regular expressions

Regular expression tutorial
/^[^<]*(<.+>)[^>]*$/  
Start of a regular expression literal

/^[^<]*(<.+>)[^>]*$/  
Anchor for beginning of line

/^[^<]*(<.+>)[^>]*$/  
Character class contain everything but <

/^[^<]*(<.+>)[^>]*$/  
Character class occurs zero or more times

/^[^<]*(<.+>)[^>]*$/  
Begin matching group. Accessed using m[1]

/^[^<]*(<.+>)[^>]*$/
Match a <

/^[^<]*(<.+>)[^>]*$/  
Match one or more characters (greedy)

/^[^<]*(<.+>)[^>]*$/  
Match a >

/^[^<]*(<.+>)[^>]*$/  
End of matching group

/^[^<]*(<.+>)[^>]*$/  
Match all zero or more characters that aren't >

/^[^<]*(<.+>)[^>]*$/
End of regular expression literal
